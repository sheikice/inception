Welcome to Inception

A Docker infrastructure project

This project builds a small web infrastructure using Docker Compose. Orchestrating NGINX, WordPress, and MariaDB inside isolated containers, linked by a private network, accessible only through TLS on port 443.

The stack
NGINX
Sole entrypoint. TLS 1.2/1.3 only. Reverse proxy to WordPress via FastCGI.
WordPress + php-fpm
Runs without NGINX. Communicates over port 9000 via FastCGI protocol.
MariaDB
Isolated database container. Credentials injected via Docker secrets.
Redis
Object cache for WordPress. Reduces database load on repeated queries.
lighttpd
Serves this static site. Reachable only inside the Docker network via NGINX proxy.
Key insight

Docker Compose itself is surprisingly simple, a few dozen lines of YAML orchestrate an entire infrastructure. The real complexity lives inside each container: MariaDB initialization scripts, php-fpm pool configuration, WordPress CLI automation. The services are harder than the orchestration.
